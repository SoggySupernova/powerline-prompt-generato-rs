# powerline-prompt-generato-rs
Create your own Powerline-style prompt for your terminal!
<br><br>
This project is mostly for practicing Rust but who knows, maybe somebody will find this useful.
<br><br>**Percentage of AI-generated code: ~25%**
<br> ^^^ I'm trying to get this to 0%!
## Features
* TUI with live preview
* Customize segment style, color, width, etc.
* Advanced control with conditional visibility
* Export directly to supported shells (see below)

## Supported shells
* fish

I will be expanding this list over time.

## Notes
* Using a Nerd Font is **highly recommended** for the best appearance.
